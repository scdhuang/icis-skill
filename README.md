# ICIS - 智能客户访谈系统

## 一句话介绍

ICIS (Intelligent Customer Interview System) 是销售团队的 AI 访谈助手，帮你设计客户访谈、分析访谈记录、生成 Sell-in 策略，让每次客户拜访都有备而战。

---

## 解决了什么问题

| 场景 | 痛点 | ICIS 解决方案 |
|------|------|---------------|
| 要拜访新客户 | 不知道问谁、问什么 | 自动生成访谈计划（3人+9个问题） |
| 访谈中遇到异议 | 不知道怎么回应 | 实时话术推荐（开场→异议→closing） |
| 访谈后写纪要 | 信息散乱，分析困难 | 自动提取痛点，匹配解决方案 |
| 准备方案提案 | 不知道推什么产品 | AI 机会点推荐 + ROI 测算 |
| 团队协作 | 经验难沉淀 | 案例库 + 话术库，新人快速上手 |

---

## 核心功能

### 1. 访谈设计器
输入客户信息，自动生成完整访谈计划：
- ✅ 推荐访谈对象（按优先级排序）
- ✅ 每人核心问题（3-5个）
- ✅ 预估访谈时长
- ✅ 配套材料清单

**示例：**
```
输入：家电制造企业，10亿规模，80%代工
输出：
  1. IT总监（45min）- 了解系统现状
  2. 销售总监（45min）- 了解业务痛点
  3. 供应链总监（45min）- 了解生产瓶颈
```

### 2. 分析引擎
上传访谈记录，自动分析输出：
- 🔍 痛点提取（Top 5）
- 💡 AI 机会点匹配（Top 3）
- 📊 业务影响量化
- 🎯 Sell-in 策略生成

**示例：**
```
输入：销售每天花2小时录入订单，经常录错
输出：
  痛点：订单处理效率低（严重度：高，频率：每天）
  解决方案：智能订单处理
  价值：2天→2小时，准确率99%+
  ROI：6个月回本
```

### 3. 话术库
覆盖全流程的销售话术：
- 🎬 开场白（3种角色）
- 💬 异议处理（5种常见异议）
- 🏁 Closing 话术（3种角色）
- 🎯 价值陈述（按痛点匹配）

**示例：**
```
异议："我们已经有ERP了"
话术："ERP解决的是流程问题，飞书+AI解决的是效率问题。
      比如订单处理，ERP需要人工录入，飞书AI可以自动识别录入。
      不是替换，是增强。"
```

### 4. 知识库
- 📚 行业模板（制造业：家电/电子/机械）
- 📖 成功案例（痛点+解决方案+效果）
- 📝 最佳实践（访谈技巧、注意事项）

## 使用场景

### 场景1：新客户拜访前
**传统方式：** 花2小时查资料、问同事、整理问题  
**用 ICIS：** 30秒输入客户信息，自动生成访谈计划

### 场景2：访谈现场
**传统方式：** 遇到异议卡壳，现场想回应  
**用 ICIS：** 实时话术推荐，从容应对

### 场景3：访谈后整理
**传统方式：** 花1小时写纪要、分析痛点、想方案  
**用 ICIS：** 粘贴访谈记录，5分钟出分析报告

### 场景4：团队协作
**传统方式：** 老员工经验在脑子里，新人摸索半年  
**用 ICIS：** 案例库共享，新人1周上手

## 快速开始

### 方式1：交互式（推荐）
直接告诉你的 AI 助手：
> "我要访谈一家家电制造企业，10亿规模，主要代工，帮我生成访谈计划"

### 方式2：命令行
```bash
# 生成访谈计划
icis plan --company "XX吸尘器" --industry appliances --scale "10亿/2000人"

# 分析访谈记录
icis analyze --input interview.txt --output report.md

# 查看话术
icis script --type objection --context "已经有ERP了"
```
